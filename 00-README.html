This directory may eventually contain RPMS to automatically install all the dependencies necessary to run OMERO. For now it is a work in progress.

At present the only useful packages are:

  • python-tables-2.4.0-1.ome6.x86_64.rpm

which depends on

  • python-numexpr-1.4.2-1.ome6.x86_64.rpm

Download these two RPMs, and run

yum localinstall python-tables-2.4.0-1.ome6.x86_64.rpm python-numexpr-1.4.2-1.ome6.x86_64.rpm

to automatically install any dependencies from the standard CentOS and EPEL repositories.
